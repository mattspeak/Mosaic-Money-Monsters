# **Structure**

The application follows a level-based progression system with character
guides throughout.

- Introduction test (to determine starting level) - 15 questions

- Level 1: Money Trailblazer (7 mini-lessons) - Basic money concepts

- Level 2: Money Scout (10 mini-lessons) - Financial terminology and
  banking

- Level 3: World Explorer (8 mini-lessons) - Currency and global finance

- Level 4: Financial Pathfinder (10 mini-lessons) - Budgeting strategies

- Level 5: Wealth Wanderer (8 mini-lessons) - Savings and banking

- Level 6: Credit Conqueror (8 mini-lessons) - Credit and debt
  management

- Level 7: Prosperity Pioneer (8 mini-lessons) - Financial planning

- Level 8: Investment Explorer (8 mini-lessons) - Introduction to
  investing

# **Age Suitability**

- Level 1: Ages 12+ (Key Stage 3)

- Level 2: Ages 14+ (Key Stage 4)

- Levels 3-8: Ages 16+ (Key Stage 5 and beyond)

# **Format Standardisation (Key Change)**

All levels use four question types:

- True/False - Users determine if a statement is true or false

- Multiple Choice - Users select one correct answer from options

- Multiple Select - Users select all correct options (where indicated)

- Match Pairs - Users match items to correct categories

**Removed formats:\**
Fill in the Blank and Sequence Order questions have been systematically
removed from all levels and replaced with compliant formats maintaining
the same learning objectives.

# **Introduction Test**

Purpose: To determine appropriate starting level for each user

- 15 questions covering all major themes across all levels

- Uses all four approved question formats

- Provides adaptive level placement based on performance

- Includes encouraging feedback throughout

- Levels: Places users in Level 1, 2, 3, or 4+ based on results
